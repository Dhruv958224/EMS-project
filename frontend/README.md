# Employee Management System (EMS)

A full-stack web application to manage employee records, built with **MERN stack** (MongoDB, Express.js, React.js, Node.js) and styled with **Tailwind CSS**.  
This project demonstrates enterprise-level features like secure authentication, modular architecture, and dynamic dashboards.

---

## 🚀 Features
- User authentication (login/logout with secure sessions)
- Role-based dashboards (Admin, HR, Employee)
- CRUD operations for employee records
- Data visualization for employee statistics
- Responsive UI with Tailwind CSS
- RESTful APIs for backend communication

---

## 🛠 Tech Stack
**Frontend:** React.js, Vite, Tailwind CSS  
**Backend:** Node.js, Express.js, MongoDB  
**Other Tools:** Git, Docker (optional), ESLint
